# 🦅 Sphynx: AI-Native Personal DevOps Station

> **「當 AI 理解了你的痛點和最終願景，它寫出的就不再是作業，而是專業的解決方案。」**

## 🌟 專案核心目的與背景脈絡
隨著 Claude Code、Gemini CLI 等本地 AI 工具的普及，開發者已能讓 AI 直接操作本地檔案與執行編譯。然而，傳統終端機的侷限性以及需要長時間守在電腦前等待任務完成，成了新的開發痛點。

**Sphynx** 的誕生就是為了打破這些限制，提供以下核心價值：
1. **專業級 UI 體驗**：透過 WebView2 與 xterm.js，讓本地操作不再枯燥。
2. **突破空間限制**：整合 Telegram Bot，實現手機遠端遙控辦公室電腦執行 AI 任務。
3. **無人值守工作流 (Fire-and-Forget)**：AI 執行完畢後自動推播通知，徹底解放開發者的專注力。

## 🛠️ 技術堆疊
- **Framework**: C# WinForms (.NET 9.0)
- **Shell Engine**: PowerShell Core (`pwsh.exe`)
- **Remote Bridge**: `Telegram.Bot` (Long Polling)
- **Terminal UI**: `Microsoft.Web.WebView2` + `xterm.js`
- **Execution Mode**: 
  - Gemini: Persistent Shell Pipe
  - Claude: Context-Aware RunOnce Process (Bypassing TTY limits)
